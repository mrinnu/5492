 # Calorie Tracker Web Application

A modern, responsive web application for tracking daily calorie intake with a beautiful glass-morphism UI design.

[Live Demo](https://curious-griffin-b6aade.netlify.app/)

## Features

- 🔐 User Authentication System
  - Secure Login/Register
  - Protected Routes
  - Session Management

- 📊 Calorie Tracking
  - Visual Calorie Charts
  - Progress Bars
  - Detailed Breakdown
  - Real-time Updates

- 🎨 Modern UI/UX
  - Glass Morphism Design
  - Responsive Layout
  - Interactive Elements
  - Smooth Animations

- 📱 Cross-Platform
  - Desktop Optimized
  - Mobile Responsive
  - Tablet Friendly

## Deployment

The application is deployed on Netlify and can be accessed at:

- Chart.js
